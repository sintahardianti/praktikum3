Fungsi cetak secara default diakhiri dengan garis baru atau enter.

```

print('C', end='')
print()
print('X')
print('Y')
print('Z')
```

### - Penggunaan Seperator

Bisa untuk memberikan perintah koma, spasi, non spasi, titik dua, dan strip.

```
w, x, y, z = 10, 15, 20, 25
print(w, x, y, z)
print(w, x, y, z, sep=',')
print(w, x, y, z, sep='')
print(w, x, y, z, sep=':')
print(w, x, y, z, sep='-----')
```
